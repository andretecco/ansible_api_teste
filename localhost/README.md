Role Localhost
=========

Role para configuração e instalação de um host local.
A configuração e instalação é feita no mesmo host que está sendo executado o Ansible.

Requisitos
------------

- SO: Ubuntu 20.04
- Ansible: 2.9
- Execução com previlégios de super-usuário (root)
